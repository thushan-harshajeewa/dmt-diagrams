## Opening the files

Recommended options to view and edit these diagrams:

- Use the desktop app: download draw.io / diagrams.net desktop and open files directly.
- Use the web app: open https://app.diagrams.net and choose **File â†’ Open from Device** or open from OneDrive if this repo is mounted.
- Many editors and IDEs have Draw.io integrations if you prefer an in-editor experience.

## Contributing

- Add new diagrams as `.drawio` files placed at repository root or an organized subfolder.
- Use descriptive filenames and avoid spaces where possible (this repo currently contains some spaces).
- Submit PRs for updates or open issues to request changes.
